# Introduction

---

Vite로 생성된 React 프로젝트의 파일 구조와 각 파일의 역할을 알아봅니다.

# 1. 프로젝트 구조

---

```
my-react-app/
├── node_modules/        # 설치된 패키지들
├── public/              # 정적 파일들
├── src/                 # 소스 코드
│   ├── App.tsx          # 메인 컴포넌트
│   ├── App.css          # App 스타일
│   ├── main.tsx         # 앱 진입점
│   └── index.css        # 전역 스타일
├── index.html           # HTML 템플릿
├── package.json         # 프로젝트 설정
├── package-lock.json    # 버전 잠금
├── tsconfig.json        # TypeScript 설정
└── vite.config.ts       # Vite 설정
```

# 2. package.json

---

프로젝트의 **신분증** 역할을 합니다.

```json
{
  "name": "my-react-app",
  "scripts": {
    "dev": "vite",
    "build": "tsc -b && vite build",
    "preview": "vite preview"
  },
  "dependencies": {
    "react": "^18.3.1",
    "react-dom": "^18.3.1"
  },
  "devDependencies": {
    "vite": "^6.0.5",
    "typescript": "~5.6.2"
  }
}
```

| 항목 | 설명 |
|------|------|
| `scripts` | 실행 가능한 명령어들 |
| `dependencies` | 앱 실행에 필요한 패키지 |
| `devDependencies` | 개발에만 필요한 패키지 |

# 3. index.html

---

앱의 **시작점** HTML 파일입니다.

```html
<!doctype html>
<html lang="en">
  <head>
    <title>Vite + React</title>
  </head>
  <body>
    <div id="root"></div>
    <script type="module" src="/src/main.tsx"></script>
  </body>
</html>
```

- `<div id="root">`: React 앱이 렌더링될 위치
- `<script src="/src/main.tsx">`: JavaScript 진입점

# 4. src/main.tsx

---

JavaScript **진입점** 파일입니다.

```tsx
import { StrictMode } from 'react'
import { createRoot } from 'react-dom/client'
import './index.css'
import App from './App.tsx'

createRoot(document.getElementById('root')!).render(
  <StrictMode>
    <App />
  </StrictMode>,
)
```

| 코드 | 설명 |
|------|------|
| `createRoot()` | React 18의 렌더링 API |
| `getElementById('root')!` | root 요소 찾기 (`!`는 반드시 존재함을 의미) |
| `<StrictMode>` | 개발 모드 추가 검사 |
| `<App />` | 메인 컴포넌트 렌더링 |

# 5. src/App.tsx

---

**메인 컴포넌트** 파일입니다.

```tsx
import { useState } from 'react'

function App() {
  const [count, setCount] = useState(0)

  return (
    <div>
      <h1>Vite + React</h1>
      <button onClick={() => setCount(count + 1)}>
        count is {count}
      </button>
    </div>
  )
}

export default App
```

| 요소 | 설명 |
|------|------|
| `function App()` | 함수형 컴포넌트 (이름은 대문자) |
| `useState` | 상태 관리 훅 |
| `return (...)` | JSX 반환 (UI 정의) |
| `export default` | 다른 파일에서 import 가능 |
